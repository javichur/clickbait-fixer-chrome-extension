Clickbait Fixer - Chrome extension

It detects and fixes clickbait links with AI locally!

You can use the Extension icon (popup) to scan all the links in the current web and get the Clickbait score for that web; or just click right over a link and select the Clickbait fixer option in the 'context menu' to analyze only that link.
